# com.ikeiwa.better-player-sync by Ikeiwa

# How To Use:
- Spawn prefab using the tab : "Tools/BetterPlayerSync/Add To Scene"
- Call BetterPlayerSync.ActivateSync() in your script or check "Auto Activate" on the prefab
- That's it

# Warning!
There are some restrictions with this system.
For example you can't use VRChat Teleport, you need to use `BetterPlayerSync.Find().Teleport()` instead
